# Feature Testing SpringBoot Web Applications with Cucumber, FluentLenium and PhantomJS

## Requirements

- Java 8
- Maven 3
- Spring 4
- PhantomJS 2

## Steps

Install Java
Install maven
Create Pom
Add cucumber
Write feature
Create cucumber runner
Integrate with fluentlium
Stand up web app
Hit page see message
Random port
